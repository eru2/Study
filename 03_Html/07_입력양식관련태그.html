폼 관련태그

form태그내에 submit버튼 클리시 해당 form안에 작성된
사용자가 입력한 값들을 서버로 넘기면서 요청하는 역할을 수행

form action="" method=""

*action : 해당 form에 사용자가 입력한 값들을 전달하면서 요청을 보낼 서버의 경로
*method : 요청 전송방식을 지정하는 속성(get(기본값), post)
> get : 요청시 사용자가 입력한 값들을 head에 담아 전송한다 -> url에 값이 전부 노출
> post : 요청시 사용자가 입력한 값들을 url에 노출되지 않는 형식으로 전달하면서

form내에 sumbit버튼 클릭시 action에 지정한 서버로 사용자가 입력한 값들이
넘어가면서 요청됨

selectbox

무조건 하나를 선택해야 할 때

radio button

selectbox처럼 여러개 중에 하나를 선책할 때 주로 사용
name속성을 통해서 여러개의 라디오 버튼을 하나의 선택지로 묶을 수 있음
checked속성을 넣은 라디오버튼이 기본값으로 설정됨
id - for송성을 통해서 input과 label태그를 묶을 수 있음
서버에 실재로 전달될 대 key - name속성값, value -> value속성값으로 전달된다.

checkbox

여러개를 선택할 대 사용

다중 선택시 주로 사용함
name 속성을 통해서 여러개의 체크박스를 하나의 선택지로 묶을 수 있음
checked속성을 통해서 체크박스를 체크된 상태로 변경될 수 있음
id - for송성을 통해서 input과 label태그를 묶을 수 있음
서버에 실재로 전달될 대 key - name속성값, value -> value속성값으로 전달된다.
